Start Socket Printing Process

In this step, you will prepare the model for printing and bring the file to the 3D printer.

Open MakerBot Desktop

Open MakerBot Desktop (represented by a red icon with a large, white "M" on it).

Choose Relevant Printer

In the "Devices" menu, use the "Select Type of Device" option, and click the item relevant to your 3D printer (either a Replicator Z18 or a Replicator 2).

Device Settings

In the "SETTINGS" menu, select the appropriate profile. For a Replicator 2, select [BLA], for a Z18, select "Z18CorsuV1." This profile includes two (2) shells, 20% infill, 0.20mm layer height, [CAN'T READ THE REST OF THE SENTENCE. MATT, HELP!]. Click "Save Settings" to apply the profile.

Import Socket Obj

From the "File" menu, use the "Add" option to import your .obj file. Select the .obj file you exported from MeshMixer and click "Open."

Position Socket Model

Using the "Move" and "Rotate" tools, position the socket, connector-down, on the print bed. Use the "On Platform" button to make sure that the bottom of the connector is touching the platform.

Generate Build File

Use the "EXPORT PRINT FILE" button to generate a build file. Once the file has been prepared, press the "Export Now" button to save an .x3g file. This is the file you will use to print the socket.
Note: The Export process "slices" the model and generates machine code that controls the 3D printer. This process can take some time. If the progress bar does not appear to be moving, reset Makerbot Desktop and start again. Once the .x3g file has been prepared and saved, it can be put onto an SD card and used in the 3D printer.
