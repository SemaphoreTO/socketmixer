...you will generate the socket from the now rectified positive model. Start by using the selection tool to trace the trim line for the socket as marked during the sca nning process, Make sure to trace above the line rather than directly on the line and verify that the line is continuous and that there are no gaps.

Start Trim Line Process

In the following steps, you will select an area on the model and use it to form a trim line.

Select Brush Size

Use the slider to select an appropriate brush size. It shouldn't be too large, but should be large enough to create a smooth, consistent selection with no breaks.

Draw Trim Line and Smooth Boundary

Use the brush to draw a trim line on the model, starting at the bottom of the patella, rising higher on the sides of the leg, and meeting in a straight line on the back of the model. To erase unwanted selections, hold down the "Ctrl" button on your keyboard while drawing. Pressing the "Accept" button will smooth your trim line.

To generate a socket, double-click anywhere below the trim line. The area on which you've double clicked should turn orange. If an area does not turn orange, simply double-click in it as well. When the entire area below the trim line is selected, press "Generate Socket" to create the socket.

Finished

Trim Line

To generate a socket, double-click anywhere below the trim line to select the appropriate part of the model. Then, press generate socket to create the socket. You can set the space between the residual limb and the socket using the ‘inner dimension’ slider and the thickness of the socket wall using the ‘outer dimension’ slider.

Start Socket Generation Process

In the following steps, you will create a socket, set its distance from the limb, and determine its thickness.

Offset Socket

Use the "Distance" slider to choose how much space there will be between the limb and the inside of the socket. Measurements are in millimetres. Press "Accept" to generate the socket.

Start Cleanup Process

In the following steps, you will smooth and remesh the socket.

Remesh

Use the "Remesh" and "Smooth" sliders to smooth the socket. Setting a high "Remesh" value will make the socket look more natural when printed.

Smooth

In this step, you can smooth the socket further. Don't worry if the trimline looks slightly ragged at this point. You'll perform an additional, manual smooth in the next step.

Trim Line Smoothing

Use the "Flatten Smooth" sculpting brush to smooth out your trim line. Simply Draw on the trimline, smoothing, until the its smoothness is satisfactory.
