Preparing


Step 4


Open the Skanect. You will see the main board on Prepare > New. Make sure the 3D Scan- ner is turned on and working properly, if so you should be able to see the Sensor in green colour. Make sure as well the GPU is available and selected, if so the icon should be in green olour.

Step 5


Now, select the specific configurations for Skanect:

Scene: Object
Bounding Box: 0.6 x 0.6 x 0.6m
Aspect Ratio: Normal

Step 6


It is also necessary to select the path where the scanning will be saved. For this, press the Blue Box beside the Path . For example, for scanning Maria we will add this path:

3D Scanning > 2014.06.11 Maria > 1st Scan

Once you have completed and rechecked all information, press Start to Start Scanning.
