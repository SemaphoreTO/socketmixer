Processing


Step 10


Now click on the Process Tab. On the left side, click on:
Colorize > Impaint Colorless: On > Run

The colorful model should have reliable colors as similar as possible.

Sharing


Step 11


Finally, click on the Share Tab . Before exporting the model we should save it. So, click on the Save Button.

And now, to export the model, click on:
Export Model > Format: OBJ, Scale: Milimeters > Export

Type the name you want for the file and Save it.
